## Kuro
A self-hosted PaaS alternative to Vercel and Heroku

> [!WARNING]
> Kuro is currently in development and may not be suitable for production use.

## Installation
Download a binary release for your platform.


**Windows**

**Linux/MacOS**

## Build
```sh 
just build-all
```

## Contributing

1. **Fork** the repository.  
2. **Create a branch** for your feature or bug fix:  
    ```bash
    git checkout -b my-feature
    ```
3. **Make your changes** and commit them with a clear message:  
    ```bash
    git commit -am "Add new feature"
    ```
4. **Push** your branch to your fork:  
    ```bash
    git push origin my-feature
    ```
5. **Open a pull request**: [https://github.com/ros-e/kuro/compare](https://github.com/ros-e/kuro/compare)
